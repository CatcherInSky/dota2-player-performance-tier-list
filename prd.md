DOTA2玩家锐评器

# 核心功能
点评队友和对手保留记录，后续匹配到会展示点评记录




# 技术框架

ts + electron + react + tailwindcss + shadcn（主要） + 数据库未定

尽量使用可商用的库如MIT协议的


# 生命周期
参考https://dev.overwolf.com/ow-electron/live-game-data-gep/supported-games/dota-2#game_state
dota2的游戏状态变化

- 游戏开始
- 在DOTA_GAMERULES_STATE_STRATEGY_TIME阶段获取到全部玩家数据之后，展示记录小窗，显示每个对应玩家的历史数据
- 游戏进行中
- 游戏结束
- 在DOTA_GAMERULES_STATE_TEAM_SHOWCASE展示点评小窗，供用户点评每个玩家的行为
- 使用快捷键alt+~可以控制主应用的显示和关闭


# 数据
有三个维度的核心数据表，下面只列出必须的数据，其他配套数据也可以记录进来
## 比赛
每一把比赛后，比赛id、所有玩家的位置、英雄、战绩、比赛胜负记录、比赛时间、比赛类型、当前版本等数据
## 玩家
对每一个玩家，玩家id、昵称、曾用名列表、当前头像、点评记录id列表
## 点评
对每一条点评，包括比赛id、玩家id、点评分数、点评文案



# 页面
程序启动时任务栏右下角有图标
右键菜单选项有
打开dota - 打开dota
打开 - 打开主页
数据 - 打开数据页
设置 - 打开设置页
退出 - 退出程序

所有页面都是使用overwolf的overlay技术悬浮在dota2应用上层但不影响操作

## 主应用
## 主页
如果在游戏中，展示当前队友和对手的历史点评信息
如果不在游戏中，展示自己的历史战绩

### 数据
一个table分三个tab展示以上三个数据表，有常规的分页器和筛选项

### 设置页
设置隐藏展示的快捷键 - 默认alt+~
导出数据按钮 - 以excel形式导出数据
导入数据按钮 - 可以导入excel数据到数据库
删除所有数据
语言选项 支持中文和英文
打分文案
从五星到一星，默认文案是夯 顶级 普通人 NPC 拉，每个文案都可以自定义

## 点评小窗
